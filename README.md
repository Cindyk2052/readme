## Getting Started

### Instalación

1. Clona el repositorio
   ```sh
   git clone https://github.com/DanielGuachamin/proyecto-movil-1erbi.git
   ```
2. Instala las dependencias del proyecto
  	```sh
  	npm install 
  	```
 3. Levanta el servidor
 	```sh
  	npm install 
  	```
